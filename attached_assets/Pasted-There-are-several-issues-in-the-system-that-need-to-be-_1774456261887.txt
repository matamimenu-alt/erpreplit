There are several issues in the system that need to be fixed and improved.
1. Numeric Input Problem
There is a bug when entering numbers in the input fields.
When I try to type a number, the field refreshes or loses focus after typing one digit, which prevents me from entering the full number (for example 1500 or 25000).
Required Fix:
•	Allow users to type the full number in the input field without interruption. 
•	Prevent any auto refresh or component re-render while typing. 
•	Ensure the numeric fields accept multi-digit numbers smoothly. 
•	Keep the cursor focus in the field until the user finishes typing.
